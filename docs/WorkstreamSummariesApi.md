# **workstream_summaries_batch**
> WorkstreamSummariesBatchOutput workstream_summaries_batch(workstream_summaries_batch_input, transferables=transferables)

/workstream_summaries/batch [POST]

Batch fetch workstream summaries by providing a list of UUIDs.
This endpoint allows for efficient retrieval of multiple workstream summaries in a single request.
The response will include successfully fetched summaries and a list of any UUIDs that were not found.
Note: If a UUID in the batch doesn't exist, it will be caught and added to the notFound list in the response.
The transferables query parameter defaults to false for performance optimization.
